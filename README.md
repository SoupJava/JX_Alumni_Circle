# 1 开发背景

随着信息技术的迅猛进步和社交媒体的普及，人们对社交网络的需求日益旺盛，尤其在
高等教育领域表现得尤为突出。校友社交 APP 作为学生在日常生活中频繁使用的软件之一，
其定位不应仅限于校内交友，更应围绕服务学生这一核心宗旨，通过功能的多样化和个性化
来更好地贴合学生的实际需求，进而吸引学生主动参与社交活动。然而，当前市面上的多数
校友社交 APP 却普遍存在功能相对单一、实用性不足以及个性化程度不够等问题，未能充
分发挥其服务大学生的独特优势。

同时，国家教育部为规范普通高等学校学生管理行为，维护普通高等学校正常的教育教
学秩序和生活秩序印发了《普通高等学校学生管理规定》，要求学生应当自觉遵守公民道德
规范，自觉遵守学校管理制度，创造和维护文明、整洁、优美、安全的学习和生活环境。
因此，我们有必要对这类 APP 进行深入研究和改进，以满足学生日益增长的需求。

# 2 作品简介

为了打造一个功能丰富、个性鲜明的社交平台，并积极响应国家教育部的号召。该项目
巧妙地结合了微信小程序和微信公众号的优势，不仅涵盖了校园动态分享、问答求助等社交
平台的基础功能，还增添了日常笔记速记、班级管理收集等实用工具，全方位满足大学生的
需求。

更为亮眼的是，该项目还引入了先进的 AI 大语言模型，构建了智慧树，为平台注入了
强大的智能元素。此外，该项目还特别设计了一个虚拟助手“小新”，它不仅能够提升平台的
社交性、定制性和实用性，更能为大学生的日常生活提供全方位的贴心提醒。使得大学校园
生活更加丰富多彩，充满无限可能。

# 3 功能分析

本小程序的主要功能如下：

1. 社交功能：校园动态，问答求助，二手出售，失物招领模块，发布，点赞，评论，私聊，
摇一摇发现同校校友等。

2. 我的课表：支持课表的创建修改删除，并被小新发现统计做出提醒或微信消息通知提醒，
支持公众号查询。

3. 我的计划：支持计划的创建修改删除，包括超时，已完成未完成处理，并被小新发现统
计做出提醒或微信消息通知提醒，支持公众号查询。

4. 随笔笔记：顺应快节奏的大学学习生活，给学生提供随手快速记笔记的功能，并接入百
度 OCR 支持扫描识别文字来达到速记效果。支持公众号查询。

5. 倒数日：支持计划的创建修改删除，并被小新发现统计做出提醒或微信消息通知提醒，
支持公众号查询。

6. 番茄钟：同过选择专注任务来进行专注，支持倒计时，提醒，记录，数据分析功能，并
被小新发现做出提醒。

7. 班级服务：支持创建修改加入班级，并进行班内的任务发布，作业提交收集打包下载等。
    
8. 文件转换：支持多种类型的文件进行转换。
    
9. 智慧树：支持浇水打卡，内部接百度文心一言 AI 大语言模型，支持异步式智能问答。
    
10. 运动：支持多种运动记录，分析，保存等功能，并被小新发现做出提醒。
    
11. 小新：通过设置小新主题颜色，表情等来同步自己的空间，聊天界面，名片等主题，同
时小新可以随机主动对自己各模块进行收集分析并提醒。

12. 公众号：支持对小程序的快捷进入，定期发布推文，以及部分数据的连接查询。

# 4 项目创新点
问答求助，失物招领的主动推送设计

当前发现的问题：

传统的校园交友小程序这些模块依然采用用户主动刻意查阅才能看到，然而在日常使用
中，用户除了自己丢失物品等基本不会对这些模块进行查阅。

解决方案：

为了解决这一问题，提高用户参加问答求助或留意失物招领的概率，我们采用主动推送
的方式，小程序将对同校校友随机主动推送最近的问答求助或者失物招领到微信模板消息上
以此来提醒用户，从而提高问题回答率和失物找回率。

虚拟角色“小新”设计

（1）形象设计：
小新主题设置后会体现在用户的个人名片，聊天页面，个人空间等多个页面供他人展示，
同时拥有多种表情和主题颜色设计。

（2）功能设计：
当用户使用小程序时，小新会主动随机获取用户各模块的信息进行处理分析，并做出随
机智能提醒。

智慧树设计

（1）浇水打卡
智慧树随机时间变渴，并提醒用户浇水打卡记录。

（2）大语言模型智能聊天
跟其他智能聊天不同的是，该智能聊天通过消息队列来进行顺序式异步调用大模型接口
（文心一言），从而实现一问一答形式，减少回答等待时间。
